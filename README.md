# 🚀 Sistema de Clientes (Mini CRM)

Sistema web completo para gerenciamento de clientes, desenvolvido com foco em organização de leads e acompanhamento de conversão.

---

## 📌 Sobre o projeto

Este projeto simula um sistema real utilizado por pequenos negócios para controle de clientes, permitindo acompanhar o progresso desde o primeiro contato até a conversão.

Além do CRUD completo, o sistema apresenta métricas que auxiliam na tomada de decisão.

---

## ✨ Funcionalidades

- ✅ Cadastro de clientes
- 📋 Listagem dinâmica
- ✏️ Edição de dados
- ❌ Exclusão de clientes
- 🔍 Busca por nome, email ou status
- 🏷️ Classificação por status:
  - Lead
  - Em contato
  - Cliente

---

## 📊 Dashboard

O sistema conta com um painel de métricas em tempo real:

- Total de clientes
- Leads
- Em contato
- Clientes convertidos
- 📈 Taxa de conversão

---

## 🛠️ Tecnologias utilizadas

### Frontend
- HTML
- CSS
- JavaScript (Vanilla)

### Backend
- Node.js
- Express

### Banco de Dados
- MySQL

---

## ⚙️ Como executar o projeto

# Clone o repositório
git clone https://github.com/rafammassa/sistema-clientes

# Acesse a pasta backend
cd sistema-clientes/backend

# Instale as dependências
npm install

# Inicie o servidor
npm start

A aplicação estará disponível em:
http://localhost:3000

## 🧠 Aprendizados

Durante o desenvolvimento deste projeto, foram trabalhados:

- Integração entre frontend e backend  
- Consumo de API com `fetch`  
- Manipulação de DOM  
- Estruturação de rotas com Node.js e Express  
- Conexão com banco de dados MySQL  
- Organização de código em um projeto full stack

Além do CRUD completo, o sistema inclui métricas de conversão, aproximando a aplicação de um cenário real de negócio.
